# Projeto Vue - Alura Tracker

Aplicação construída com **Vue 3** para praticar componentes, organização de interface e fluxo de desenvolvimento com Vue CLI. O projeto funciona como um estudo prático de front-end moderno com TypeScript, lint e build de produção.

## Tecnologias

- Vue 3
- TypeScript
- Vue CLI
- ESLint
- Font Awesome
- HTML/CSS

## Como executar

Instale as dependências:

```bash
npm install
```

Rode em modo desenvolvimento:

```bash
npm run serve
```

Gere a versão de produção:

```bash
npm run build
```

Execute o lint:

```bash
npm run lint
```

## O que este projeto demonstra

- Estrutura de aplicação Vue.
- Componentização de telas e elementos de interface.
- Uso de TypeScript em projeto front-end.
- Configuração de build, lint e ambiente local.
